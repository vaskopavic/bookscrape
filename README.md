# Bookscrape Support - a full-stack customer support chatbot

Bookscrape is a customer support chatbot that offers real-time responses to customer inquiries. It is built with TypeScript and TailwindCSS, which ensure a robust and reliable user experience. Additionally, Bookscrape features secured, rate-limited API routes, providing maximum privacy and safety for users.

Features:

- Real-time streamed chatbot responses
- Optimistic updates for the best UX
- Secured, rate-limited API routes to prevent abuse
- Built with TypeScript
- TailwindCSS
- Icons from Lucide
- Class merging with tailwind-merge
- Conditional classes with clsx

### Prerequisites

**Node version 14.x**


### Install packages

```shell
yarn install
```

### Setup .env file

```js
OPENAI_API_KEY=
REDIS_URL=
REDIS_SECRET=
```

### Start the app

```shell
yarn dev
```
